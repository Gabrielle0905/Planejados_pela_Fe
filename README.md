# **Planejados pela Fé**
Sistema de Cadastro dos Encontros das Mães Que Oram pelos Filhos

<hr>
## Sobre o Projeto
>Planejados pela Fé é um sistema desenvolvido para organizar e registrar encontros de grupos de oração de uma forma prática, sendo possível registrar datas, participantes e informações importantes de cada encontro.
>O tema “mães que oram pelos filhos” foi utilizado apenas para fins acadêmicos e não representa nenhum grupo real ou oficial. Ele serve apenas como base conceitual para demonstrar as funcionalidades solicitadas na disciplina.

<hr>
## Objetivo
>Este projeto foi desenvolvido como requisito avaliativo da disciplina Linguagem de Programação, com foco na construção de um sistema funcional utilizando boas práticas de organização, estruturação e registro de dados.
